# StationConfig 필드 설명

## 각 필드가 실제로 표시되는 위치

```typescript
{
  name: 'KBS 1라디오',           // 방송국 이름
  streamUrl: 'https://...',      // 스트림 주소
  category: 'KBS',               // 카테고리
  artwork: 'https://로고.png',   // 썸네일 이미지
  description: '설명',           // 방송국 설명
}
```

---

## 📱 1. name (방송국 이름)

### 표시되는 곳:

#### ✅ 홈 화면 - 방송국 카드
```
┌─────────────────┐
│   [썸네일 이미지]  │
├─────────────────┤
│ KBS 1라디오     │ ← name이 여기 표시
│ KBS             │
└─────────────────┘
```

#### ✅ 미디어 컨트롤 (알림)
```
Android 알림창:
┌──────────────────────────┐
│ [썸네일]  KBS 1라디오      │ ← name이 여기 표시
│           Live Radio      │
│   ⏮  ⏸  ⏭               │
└──────────────────────────┘
```

#### ✅ 잠금화면
```
잠금화면 미디어 컨트롤:
KBS 1라디오          ← name이 여기 표시
Live Radio
[이전] [일시정지] [다음]
```

#### ✅ 플레이어 화면
```
┌──────────────────┐
│                  │
│   [큰 썸네일]     │
│                  │
│  KBS 1라디오     │ ← name이 여기 표시
│  Live Radio      │
└──────────────────┘
```

---

## 🔗 2. streamUrl (스트림 주소)

### 사용되는 곳:

#### ✅ 백그라운드 (사용자에게 보이지 않음)
- TrackPlayer가 이 URL로 라디오 스트림 재생
- 사용자에게는 직접 표시 안 됨
- 오류 발생 시 로그에만 표시

```typescript
// 내부 동작:
TrackPlayer.add({
  url: 'https://radio.bsod.kr/stream/?stn=kbs&ch=1radio',  ← 이 URL로 재생
  title: 'KBS 1라디오',
});
```

---

## 📂 3. category (카테고리)

### 표시되는 곳:

#### ✅ 홈 화면 - 카테고리 탭
```
┌─────────────────────────────────────┐
│ [전체] [KBS] [MBC] [SBS] [CBS]...  │ ← category로 탭 생성
└─────────────────────────────────────┘
```

#### ✅ 홈 화면 - 방송국 카드 (재생 중 아닐 때)
```
┌─────────────────┐
│   [썸네일]       │
├─────────────────┤
│ KBS 1라디오     │
│ KBS             │ ← category가 여기 표시
└─────────────────┘
```

#### ✅ 필터링 기능
- 사용자가 "KBS" 탭 선택 → category가 "KBS"인 방송국만 표시
- 사용자가 "전체" 탭 선택 → 모든 방송국 표시

## 🖼️ 4. artwork (썸네일 이미지)

### 표시되는 곳:

#### ✅ 홈 화면 - 방송국 카드
```
┌─────────────────┐
│                 │
│   [여기 표시]    │ ← artwork가 여기 표시 (정사각형)
│                 │
├─────────────────┤
│ KBS 1라디오     │
└─────────────────┘
```

#### ✅ 미디어 컨트롤 (알림)
```
Android 알림:
┌────────────────────────┐
│ [로고] KBS 1라디오      │ ← artwork가 여기 표시 (작은 크기)
│        Live Radio      │
└────────────────────────┘
```

#### ✅ 잠금화면
```
잠금화면:
┌──────────────────┐
│                  │
│   [큰 로고 이미지]│ ← artwork가 여기 표시 (큰 크기)
│                  │
│  KBS 1라디오     │
└──────────────────┘
```

#### ✅ iOS 컨트롤 센터
```
컨트롤 센터:
┌──────────────────┐
│   [앨범 커버]     │ ← artwork가 여기 표시
│  KBS 1라디오     │
│   [재생 버튼]     │
└──────────────────┘
```

## 📝 5. description (방송국 설명)

### 현재 상태: ⚠️ 미사용

현재는 저장만 되고 UI에 표시 안 됨. 향후 추가 가능:

#### 🔮 향후 사용 가능한 곳 (예시):

##### 방송국 상세 화면
```
┌──────────────────────┐
│   [큰 썸네일]         │
│                      │
│  KBS 1라디오         │
│  ───────────────    │
│  KBS 제1라디오       │ ← description이 여기 표시 가능
│                      │
│  [재생] [즐겨찾기]    │
└──────────────────────┘
```

##### 툴팁
```
방송국 카드를 길게 누르면:
┌──────────────────┐
│ KBS 제1라디오    │ ← description이 여기 표시 가능
└──────────────────┘
```

## 🎯 실제 예시

```typescript
{
  name: 'KBS 1라디오',
  streamUrl: 'https://radio.bsod.kr/stream/?stn=kbs&ch=1radio',
  category: 'KBS',
  artwork: 'https://example.com/kbs1-logo.png',
  description: 'KBS 제1라디오',
}
```

### 사용자가 보는 화면:

#### 홈 화면:
```
┌─────────────────────────┐
│ [KBS 탭 선택됨]         │
├─────────────┬───────────┤
│             │           │
│ [KBS1 로고] │[KBS2 로고]│
│             │           │
├─────────────┼───────────┤
│KBS 1라디오  │KBS 2라디오│
│KBS          │KBS        │
└─────────────┴───────────┘
```

#### 재생 중 알림:
```
┌───────────────────────────┐
│ 🔊 KRadio24               │
│                           │
│ [KBS1] KBS 1라디오        │
│        Live Radio         │
│                           │
│   ⏮️  ⏸️  ⏭️              │
└───────────────────────────┘
```

## 📊 필드 중요도

| 필드 | 필수 여부 | 사용자에게 보임 | 용도 |
|------|----------|---------------|------|
| `name` | ✅ 필수 | ⭐⭐⭐⭐⭐ | 모든 곳에 표시 |
| `streamUrl` | ✅ 필수 | ❌ 안 보임 | 재생용 |
| `category` | ✅ 필수 | ⭐⭐⭐ | 탭/필터링 |
| `artwork` | ⚠️ 선택 | ⭐⭐⭐⭐ | 썸네일 표시 |
| `description` | ⚠️ 선택 | ❌ 현재 안 보임 | 미래용 |

---

## 🎨 UI 개선 제안

현재 `description`을 활용하지 않고 있습니다. 추가할 수 있는 기능:

1. **방송국 롱프레스 시 상세 정보 표시**
2. **검색 결과에 설명 표시**
3. **플레이어 화면에 부제목으로 표시**
4. **즐겨찾기 목록에 부가 정보로 표시**
